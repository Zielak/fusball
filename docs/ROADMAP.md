# Fusball Roadmap

Tracked feature backlog for agent/human context. Check off in PRs when done.

## Done (MVP)

- [x] Top-down table with side goals
- [x] 8 sticks (4 per player), mirrored layout
- [x] Configurable pawn counts, bumpers, dimensions
- [x] Smooth keyboard slide (QWERTY columns)
- [x] Arcade ball physics vs walls + pawns
- [x] Player1 controllable; player2 static

## Next up

- [ ] Stick rotation input (keyboard and/or mouse)
- [ ] Goal scoring (EventBus event + UI)
- [ ] Ball reset after goal
- [ ] Player2 bot (simple chase/block AI)

## Later

- [ ] Fake 3D ball mechanics (spin, bank shots)
- [ ] Match rules (first to N, kickoff)
- [ ] Sound + juice (hit feedback, goal celebration)
- [ ] Sprite art replacing procedural graphics
- [ ] Matter.js physics evaluation
- [ ] Two-human local multiplayer input

## Config knobs to expose in UI (optional)

- Slide speed preset (slow / normal / fast)
- Table size preset
- Practice mode (solo, no opponent sticks)
